# zmxy

Godot 4 2D action game prototype project.

## Development setup

- Engine: Godot 4.x
- Language: GDScript (default)
- Editor: Godot + Cursor/VSCode

## Project structure

- `scenes/`: scene files
- `scripts/`: gameplay scripts
- `assets/`: art/audio resources

## Prototype controls

- Move: `A/D` or arrow keys
- Jump: `Space` (or Enter from `ui_accept`)
- Attack: `J`
